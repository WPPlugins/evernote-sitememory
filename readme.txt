WP-Evernote can add Evernote SiteMemory Buttons on the bottom of post that you selected, you can change the options of buttons quickly and easily.

== Description ==
= Main features of the plugin: =
Evernote had provided a way to clip pages of your site for the visitors, but it was  too complicated, especially for the beginners, and it must be used in every post.<br>
With this plugin, you can:<br>
1. Only display SiteMemory button in the post with specifyed tag, and you can set to display in every post.<br>
2. Edit SiteMemory settings quickly and easily.<br>
3. Don't need clip and edit the php or html code.<br>


== Installation ==
1. Unzip and upload the `wp-evernote` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Modify Plugin settings though the 'Evernote SiteMemory' menu in WordPress.

== Frequently Asked Questions ==
= What is Evernote SiteMemory? =
It's provided by Evernote, SiteMemory helps visitor to save content to their evernote, More information about [Evernote Site Memory](http://www.evernote.com/about/developer/sitememory/).
= How to display the Button in the end of posts that i selected? =
First, you should set a Control Tag in the wp-evernote, for example, "evernote", when you want display SiteMemory Button in the end of a post, add "evernote" as its tag, it will be work.
= I set control tag, and I add it to my post, but it doesn't work, why? =
Check the "Display Button or NOT" options, make sure you selected "Yes".
